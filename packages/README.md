# Pulse Packages

Shared packages for the future Pulse monorepo.

This folder is scaffold-only for now. Package tooling and exports will be added in a later step.

## Planned Packages

- `ui`: shared design system components and tokens.
- `database`: database client helpers and persistence utilities.
- `types`: shared TypeScript domain contracts.
- `config`: shared configuration and environment helpers.
- `pdf`: proposal and document rendering helpers.
